CipherScope

Fast static scanner for cryptographic libraries

Find third-party cryptographic libraries and call sites across 10 programming languages with low false positives and high performance.

Install & Run

cargo install --git https://github.com/your-org/cipherscope

Or build from source: cargo build --release

Example Outputs

Table Format (Default)

Language | Library | Count | Example ---------|---------|-------|-------- Rust | RustCrypto | 2 | src/main.rs:12 aes_gcm::Aes256Gcm Python | PyCA cryptography | 1 | auth.py:8 Fernet(key)

JSONL Format

{ "language": "Rust", "library": "RustCrypto", "file": "src/main.rs", "span": { "line": 12, "column": 5 }, "symbol": "aes_gcm::Aes256Gcm", "snippet": "use aes_gcm::Aes256Gcm;", "confidence": 0.99, "detector_id": "detector-rust" }

Patterns Configuration

[[library]] name = "PyCA cryptography" languages = ["Python"] [library.patterns] include = [ "^\\s*from\\s+cryptography\\b" ] apis = [ "\\bFernet\\(", "\\.encrypt\\(", "\\.decrypt\\(" ]

Define import patterns and API calls for each library

[[library]] name = "OpenSSL" languages = ["C", "C++"] [library.patterns] include = [ "^\\s*#\\s*include\\s*" ] apis = [ "\\bEVP_[A-Za-z0-9_]+\\s*\\(", "\\bRSA_[A-Za-z0-9_]+\\s*\\(" ]

C/C++ libraries use include patterns and function calls

Supported Languages

C C++ Java Go Rust Python PHP Swift Objective-C Kotlin

Quick Start

cryptofind . --progress

Scan current directory with progress bar

Output Formats

cryptofind . --json > findings.jsonl
